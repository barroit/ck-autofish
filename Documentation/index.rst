==========================
The mikufish Documentation
==========================

This tree documents ``mikufish``, the Core Keeper auto fishing mod that
automatically casts, hooks, and reels with zero-delay reactions.

Read the user manual if you want to configure it. Read the hacking guide if you
want to change code, package releases, or maintain the repository.
